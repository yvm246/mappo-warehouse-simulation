# BigThinkAI-Discrete-Mappo

# Multi-Agent Warehouse Coordination using MAPPO

This project implements a Multi-Agent Proximal Policy Optimization (MAPPO) reinforcement learning system to coordinate multiple agents in a warehouse grid environment.

Agents learn cooperative policies to efficiently navigate the environment and complete delivery tasks.

## Technologies
- Python
- PyTorch
- NumPy
- Matplotlib
- PettingZoo

## Features
- Multi-agent reinforcement learning
- Centralized critic training
- Agent coordination and policy optimization
- Visualization tools for agent behavior

## Project Structure

train_discrete_mappo.py – training pipeline  
mappo_models.py – neural network models  
mappo_agent.py – agent logic  
mappo_buffer.py – trajectory storage  
visualization.py – policy visualization
